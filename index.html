    The connecting IP is determined using the server variable $_SERVER['REMOTE_ADDR']. If the detected IP doesn't match yours, it means that your web server is under a proxy and you need to tweak your server to set the real connecting IP.

    For Nginx, you must use ngx_http_realip_module. For Apache, mod_remoteip.

    Make sure that you address this issue as the system relies on accurate IP detections to provide basic functionalities and to protect against spam, flooding, and brute force attacks.
